# Libreoffice Writer

## Settings

Open "User interface" under View, and change the following settings:

- UI variants: Sidebar
	- Apply to All

Change the following settings from the toolbar:

- View
	- Toolbars
		- Standard: Off (toggle if needed)
	- Status Bar: Off
	- Rulers
		- Rulers: Off
	- Scrollbars
		- Vertical Scroll Bar: Off
	- Sidebar: Off (toggle if needed)
