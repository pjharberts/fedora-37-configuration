# Live Media Installation

1. Install to Hard Drive
2. English / English (United Staes)
3. Time & Date
4. City: Chicago
5. Installation Destination
6. Storage Configuration: Custom
7. Add a new mount point
  - swap with half the capacity as the device's RAM
  - /boot with 1 GiB capacity
  - /boot/efi with 500 MiB capacity
  - / using the remaining capacity
8. Modify the / mount point's volume name and label name to the device name you plan to use
9. Begin installation
10. Finish installation
11. Shut down
12. Unplug live USB when shut down
